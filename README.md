# Guia Completo do Debian

  Guia completo do Debian em Português Brasileiro — do iniciante ao avançado com exemplos práticos.

  ## Tecnologias

  - React + Vite + TypeScript
  - Tailwind CSS + shadcn/ui
  - Hash Router (compatível com GitHub Pages)

  ## Como rodar localmente

  ```bash
  npm install
  npm run dev
  ```

  ## Deploy

  O projeto é automaticamente publicado no GitHub Pages via GitHub Actions a cada push na branch `main`.
